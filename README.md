# mock平台
## 一期功能点
* 支持项目配置根目录
* 支持url、method、description
* 支持跨域
* 支持json和yaml文件导入，在线编辑
* 支持多个response body，手动选择其中一个返回
* 支持url列表按平铺和树状结构展示
* 谷歌插件，做目录、文件的匹配

## 二期功能点
* GET POST等参数解析
* 状态码匹配
* 版本控制
* 正则匹配路径
* rest api
* response对象管理
* 等等等等
